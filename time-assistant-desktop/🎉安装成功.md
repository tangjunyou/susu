## ⚠️ 如果长时间没反应

### 如果5分钟后仍无反应：

1. **检查是否在下载依赖**
   - 查看输出是否显示 "Downloading crates"
   - 这可能需要较长时间，取决于网络速度

2. **检查是否有错误信息**
   - 查看红色的错误提示
   - 如果是链接器错误，需要安装VS Build Tools

3. **重新启动**
   ```powershell
   # 按 Ctrl+C 停止
   # 然后重新运行
   npm run tauri:dev
   ```

## 🔧 可能需要的额外步骤

### 如果出现链接器错误（linker error）

**安装Visual Studio Build Tools：**

```powershell
winget install Microsoft.VisualStudio.2022.BuildTools
```

安装时选择"使用C++的桌面开发"工作负载。
