# 🎉 超紧凑模式使用指南

## 🎯 快速开始

### 1️⃣ 开启超紧凑模式

**步骤：**
1. 点击右上角 ⚙️ 设置图标
2. 找到"窗口置顶"开关
3. 点击开启 ✅

**结果：**
- ✅ 窗口自动缩小到 448px 宽
- ✅ 标题栏消失
- ✅ 统计信息消失
- ✅ 筛选器消失
- ✅ 只显示任务卡片
- ✅ 右上角出现浮动设置按钮

---

### 2️⃣ 在超紧凑模式下操作

**可以做的事情：**
- ✅ 点击任务卡片 → 查看详情
- ✅ 点击"开始计时" → 开始任务
- ✅ 点击"停止计时" → 暂停任务
- ✅ 点击"添加反思" → 记录心得
- ✅ 勾选圆圈 → 完成任务
- ✅ 点击垃圾桶 → 删除任务
- ✅ 点击浮动按钮 ⚙️ → 打开设置

**不能做的事情：**
- ❌ 添加新计划（需要切换到完整模式）
- ❌ 查看统计数据（需要切换到完整模式）
- ❌ 筛选已完成任务（自动只显示进行中）

---

### 3️⃣ 切换回完整模式

**步骤：**
1. 点击右上角浮动的 ⚙️ 按钮
2. 关闭"窗口置顶"开关
3. ✅ 恢复完整界面

## 🎯 最佳实践

### ✅ 推荐做法

1. **工作时使用超紧凑模式**
   - 不干扰主要工作
   - 快速查看当前任务
   - 专注执行

2. **管理时使用完整模式**
   - 添加新计划
   - 查看统计数据
   - 整体规划

3. **合理放置窗口**
   - 边缘位置：不遮挡主要内容
   - 屏幕角落：方便查看
   - 次要屏幕：多屏用户

### ❌ 不推荐做法

1. **不要一直用完整模式**
   - 占用太多空间
   - 信息过载
   - 分散注意力

2. **不要把超紧凑窗口放中间**
   - 遮挡主要工作区
   - 频繁移动窗口
   - 影响效率

3. **不要频繁切换**
   - 打断工作流程
   - 降低效率
   - 增加认知负担

## 🔧 自定义技巧

### 如果觉得还是太宽

修改 `src/App.tsx`：

```typescript
// 第 44 行附近
className={`${isCompactMode ? 'max-w-sm' : 'max-w-4xl'}`}
// 从 max-w-md (448px) 改为 max-w-sm (384px)
```

### 如果觉得卡片间距太大

修改 `src/App.tsx`：

```typescript
// 第 63 行附近
<div className="p-2 space-y-2">
// 从 space-y-2 改为 space-y-1
```

### 如果想隐藏描述

修改 `src/components/PlanCard.tsx`，添加条件：

```typescript
{!isCompactMode && plan.description && (
  <p className="text-sm">{plan.description}</p>
)}
```

## 🆘 常见问题

### Q1: 怎么添加新计划？

**A:** 关闭置顶，切换到完整模式，点击"添加计划"按钮。

### Q2: 怎么查看已完成的任务？

**A:** 关闭置顶，切换到完整模式，使用筛选器选择"已完成"。

### Q3: 怎么调整窗口大小？

**A:** 超紧凑模式宽度固定为 448px，如需调整请修改代码中的 `max-w-md`。

### Q4: 可以自定义快捷键吗？

**A:** 当前版本暂不支持，未来版本会添加。

### Q5: 浮动按钮遮挡任务了怎么办？

**A:** 浮动按钮使用半透明+模糊效果，不会严重遮挡。如有需要，可以修改代码调整位置。
