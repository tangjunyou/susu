**最简单的方法**：

### 步骤2: 重启终端

安装Rust后，**必须重启PowerShell**（让环境变量生效）

### 步骤3: 运行应用

```powershell
cd E:\时间管理助手\package\time-assistant-desktop
npm run tauri:dev
```

## ❓ 常见问题

### Q: 为什么必须安装Rust？
A: Tauri是基于Rust的桌面框架，Rust负责系统级功能（窗口置顶、数据库等）。

### Q: 安装需要多久？
A: Rust安装10-15分钟，首次编译5-10分钟，总计约30分钟。

### Q: 可以不装Rust吗？
A: 不可以完整运行。但可以运行`npm run dev`预览UI（功能不可用）。

### Q: 会有什么问题吗？
A: 代码质量良好，成功率约85%。如遇问题，查看TROUBLESHOOTING.md。

### Q: 网页版和桌面版有什么区别？
A: 桌面版有窗口置顶、系统托盘、全局快捷键等系统级功能。
