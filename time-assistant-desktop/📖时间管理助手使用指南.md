# 📖 时间管理助手 - 完整使用指南

欢迎使用时间管理助手！这份指南将教你如何最有效地使用这个工具来管理你的时间和任务。

---

## 🚀 快速开始（5分钟上手）

### 第一步：启动应用

```powershell
# 进入项目目录
cd E:\时间管理助手\package\time-assistant-desktop

# 启动应用
npm run tauri:dev
```

### 第二步：添加你的第一个任务

1. 点击右上角的 **"+ 添加计划"** 按钮
2. 填写任务信息：
   - **标题**：写任务目标（如：完成项目文档）
   - **描述**：详细说明（可选）
   - **优先级**：高/中/低
   - **分类**：工作/学习/生活/健康/其他
   - **开始日期**：今天
   - **结束日期**：截止日期
3. 点击 **"添加"**

### 第三步：开始执行任务

1. 找到刚添加的任务卡片
2. 点击 **绿色播放按钮 ▶** 开始计时
3. 专注工作
4. 完成后点击 **暂停按钮 ⏸** 停止计时

✅ 恭喜！你已经开始使用时间管理助手了！

```

```

- ⏸ 变成 ▶

```
暂停状态 → 点击 ▶ 按钮 → 继续计时
```

**效果：**
- 从上次暂停的时间继续累计

### 实践4：优先级管理法（重要紧急四象限）

**分类原则：**

```
           紧急
      ┌─────┴─────┐
      │           │
重要│ 1.立即做 │ 3.安排做│
      │ [高优先] │ [中优先]│
    ──┼─────────┼─────────┤
      │           │
不重要│ 2.委托做 │ 4.不要做│
      │ [低优先] │ [删除]  │
      └───────────┘

1. 重要且紧急 → 高优先级 → 立即做
2. 不重要但紧急 → 低优先级 → 委托或快速处理
3. 重要但不紧急 → 中优先级 → 安排时间做
4. 不重要不紧急 → 删除 → 不要做
```

**应用示例：**

```
本周任务清单：

[高] 明天截止的项目报告
     → 重要且紧急，今天必须完成

[高] 客户投诉处理
     → 重要且紧急，立即处理

[中] 下周的会议准备
     → 重要但不紧急，安排明天做

[中] 学习新技能
     → 重要但不紧急，每天1小时

[低] 整理桌面
     → 不重要但紧急，工作间隙快速处理

[删除] 刷社交媒体
     → 不重要不紧急，删除任务
```

## 💡 使用技巧

### 技巧1：合理估算时间

**新手常见问题：**
- ❌ 估算1小时，实际花了3小时
- ❌ 任务太大，不知道从哪开始
- ❌ 时间安排过满，无法完成

**解决方案：**
```
1. 时间估算 × 1.5倍（预留缓冲）
2. 大任务拆分成1-2小时的小任务
3. 每天只安排6小时工作量（留2小时应对突发）
4. 使用应用追踪实际时间，逐步提高估算准确性
```

**示例：**

```
任务：完成项目文档

初始估算：2小时
调整后：2小时 × 1.5 = 3小时

拆分：
- 任务1：收集资料（1小时）
- 任务2：撰写初稿（1.5小时）
- 任务3：审阅修改（30分钟）

实际追踪：
- 任务1：实际花了1小时15分钟 ✓
- 任务2：实际花了2小时 ⚠️
- 任务3：实际花了45分钟 ✓

总结：下次写作类任务估算 × 2倍
```

### 技巧3：利用反思改进

**反思不是日记！**

```
❌ 错误示例：
"今天写了文档，挺好的。"

✅ 正确示例：
"今天完成文档初稿3000字，但数据收集环节花了1小时，
比预期多30分钟。原因是资料分散在多个文件中。
明天开始新任务前，先集中整理所有相关资料。"
```

**反思驱动改进：**

```
第1周：
反思 → 发现写作时间估算不准
改进 → 写作任务时间 × 2

第2周：
反思 → 发现早上效率最高
改进 → 重要任务安排在早上

第3周：
反思 → 发现被打断太多次
改进 → 设置"勿扰时间"

第4周：
反思 → 发现任务完成率明显提升
成果 → 从50% → 80%
```

## ⚠️ 常见问题

### Q1：我总是忘记开始计时怎么办？

**解决方案：**
```
1. 开启"窗口置顶"
2. 应用始终在最前面
3. 视觉提醒你正在做什么
4. 养成习惯：开始任务 = 点击 ▶
```

### Q2：任务太多，不知道先做哪个？

**解决方案：**
```
1. 使用优先级标签
2. 高优先级任务最多3个/天
3. 早上先做最重要的1个
4. 参考"重要紧急四象限"
```

### Q3：计时经常忘记暂停？

**解决方案：**
```
1. 设置手机闹钟提醒（25分钟/次）
2. 使用番茄工作法
3. 离开座位前检查一次
4. 查看累计时间是否异常
```

### Q4：反思不知道写什么？

**使用模板：**
```
📝 完成了什么？
🤔 遇到了什么问题？
💡 有什么收获？
🎯 如何改进？
```

### Q5：应用关闭后数据会丢失吗？

**回答：**
```
✅ 不会！所有数据保存在本地数据库
✅ 位置：C:\Users\你的用户名\AppData\Local\time-assistant\data.db
✅ 即使关闭应用，数据也会保留
✅ 建议定期备份数据库文件
```
